カスタム投稿タイプを追加し、リンクを管理するプラグインです。


== Description ==
カスタム投稿タイプ「OLM LINK」を追加し、リンクを管理できます。

<ul>
<li>コンテンツテキストからリンクを取得</li>
<li>CSVによるインポート・エクスポート</li>
<li>投稿ページに追加したメタボックスを設定し、ブログカードのようにグループリンクを出力</li>
<li>登録されているリンクのキーワード（編集ページで設定できます）にマッチするテキストをリンクテキストに変換</li>
</ul>

などが使用可能になります。


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/opt-link-manager` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress


== Frequently Asked Questions ==

= 使い方 =

「OLM Links」->「Tools」でリンクの取得などを操作できます。

「OLM Link」の各リンクページで「タイトル」「URL」「サムネイル」「グループ」などを設定し、投稿ページに追加したメタボックスでコンテンツページに反映させることができます。
